

## 내장경고창
alert(message)	단순한 메시지 알림	확인(OK)만 있음
confirm(message)	사용자의 확인/취소 선택을 받음	확인(OK) / 취소(Cancel)
prompt(message, default)	사용자로부터 텍스트 입력을 받음	입력창 + 확인 / 취소


## 외장경고창
SweetAlert2
- https://sweetalert2.github.io/
```html
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>

<script>
Swal.fire({
  title: '정말 삭제하시겠습니까?',
  text: "이 작업은 되돌릴 수 없습니다!",
  icon: 'warning',
  showCancelButton: true,
  confirmButtonText: '네, 삭제할게요',
  cancelButtonText: '아니요, 취소할래요'
}).then((result) => {
  if (result.isConfirmed) {
    // 삭제 처리
    console.log("삭제됨");
  }
});
</script>

```
