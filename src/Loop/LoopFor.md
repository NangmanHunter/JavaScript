## EnhancedForLoop

- a

Of
```js

for(let a of arr){
    console.log(a);
}
```


## ForLoop
- arr
- arr[i]
- i

In
```js
arr=[1,2,3];

for(let i in arr){
    console.log(arr[i]);
}
```

Length
```js
arr=[1,2,3];

for(let i=0; i<arr.length; i++){
    console.log(arr[i]);
}
```
